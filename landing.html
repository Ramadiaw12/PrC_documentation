Créer un compte PromptCraft prend moins de 30 secondes. Aucune carte bancaire n'est requise pour commencer.

Optimisation → Bonnes pratiques

Bonnes pratiques

Pour obtenir les meilleurs prompts, voici ce qui fait la différence dans vos descriptions initiales.

✅ À faire
  • Précisez le contexte : "pour un email de relance client B2B"
  • Indiquez le public cible : "pour un développeur junior"
  • Mentionnez le ton souhaité : "formel mais accessible"
  • Donnez un exemple de format si vous en avez un
❌ À éviter
  • Les descriptions trop vagues : "écris quelque chose sur le marketing"
  • Les instructions contradictoires dans une même demande
  • Les besoins non définis : "améliore mon texte" sans fournir le texte
