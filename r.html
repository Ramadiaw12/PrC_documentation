Documentation PromptCraft

Tout ce que vous devez savoir pour transformer vos idées en prompts professionnels — de la création de compte aux fonctionnalités avancées.

Lecture : ~12 min Mis à jour le 8 juin 2026
Bienvenue dans PromptCraft Docs
Maîtrisez l'art du prompt en quelques minutes

Cette documentation couvre l'ensemble des fonctionnalités de PromptCraft, des cas d'utilisation concrets, et les meilleures pratiques pour obtenir des résultats parfaits avec n'importe quel LLM.

🚀 Démarrage rapide 💡 Cas d'utilisation ⚡ Optimisation 🔗 API Reference

Démarrage #

PromptCraft est un agent IA qui transforme une idée brute en prompt structuré et professionnel en moins de 3 secondes. Ce guide vous emmène de zéro à votre premier prompt parfait.

Création d'un compte #

PromptCraft utilise uniquement l'authentification Google — pas de mot de passe à mémoriser.

1
Accéder à l'application
Rendez-vous sur prompt-craft26.netlify.app depuis votre navigateur.
2
Cliquer sur "Continuer avec Google"
Une fenêtre Google OAuth s'ouvre. Sélectionnez le compte souhaité.
3
Accéder au tableau de bord
Vous êtes redirigé vers votre espace. Votre plan Gratuit est activé automatiquement.
ℹ️
Données collectées PromptCraft n'accède qu'à votre nom, email et photo de profil Google. Vos prompts sont stockés de manière sécurisée.

Connexion #

Si vous possédez déjà un compte, la connexion est instantanée. Cliquez sur "Continuer avec Google" et sélectionnez votre compte — vous êtes connecté en moins de 3 secondes.

💡
Session persistante Votre session reste active pendant 30 jours. Pas besoin de vous reconnecter à chaque visite.

Première utilisation #

À votre première connexion, vous découvrez l'interface principale composée de quatre zones :

📝
Zone d'entrée
Saisissez votre idée brute ici. Parlez naturellement, sans technique.
⚡
Agent IA (3 étapes)
Analyse → Structuration → Raffinement. Le pipeline s'exécute en temps réel.
📋
Prompt généré
Copiez en un clic et collez dans ChatGPT, Claude ou Gemini.
🗂️
Historique
Retrouvez tous vos prompts précédents avec date, catégorie et score.

Création d'un premier prompt #

1
Saisir votre besoin
Exemple : "Je veux un assistant pour rédiger des emails professionnels en anglais"
2
Cliquer sur "Générer"
Vous voyez les 3 étapes défiler en temps réel : analyse, structuration, raffinement.
3
Copier et utiliser
Cliquez sur "Copier" et collez votre prompt structuré directement dans votre outil IA.
✦ Entrée brute → Prompt généré
Votre idée
Je veux un assistant pour rédiger des emails pro en anglais
Prompt structuré
RÔLE : You are an expert business communication specialist with 10+ years in corporate email writing...

TÂCHE : Draft professional emails that are formal yet warm, clear and action-oriented...

FORMAT : Subject line + 3 paragraphs max + clear call-to-action + professional closing...

Gestion des prompts #

PromptCraft vous offre un contrôle total sur vos prompts : création, modification, suppression et organisation.

Créer un prompt

Deux approches possibles :

  • Génération IA — Saisissez votre idée et laissez l'agent structurer automatiquement (recommandé)
  • Création manuelle — Cliquez sur "+ Nouveau prompt" pour rédiger directement les 5 sections
✨
Astuce Combinez les deux approches : générez automatiquement puis affinez manuellement section par section.

Modifier un prompt

  1. Ouvrez votre Historique depuis la barre latérale
  2. Cliquez sur le prompt à modifier
  3. Sélectionnez "Éditer" (icône crayon)
  4. Modifiez les sections souhaitées et sauvegardez
ℹ️
Chaque modification crée une nouvelle version. L'historique de versions est accessible via l'icône 🕐 sur chaque prompt.

Supprimer un prompt

Survolez un prompt dans l'historique et cliquez sur l'icône 🗑️. Une confirmation est demandée avant suppression.

⚠️
Suppression définitive Les prompts supprimés ne peuvent pas être récupérés. Exportez vos prompts importants avant suppression Pro

Bibliothèque de prompts #

La bibliothèque est votre espace personnel de prompts organisés et retrouvables en quelques secondes.

Recherche

La barre de recherche (⌘ + K) effectue une recherche full-text dans vos prompts : titre, contenu, tags et catégories. Les résultats s'affichent en temps réel.

Catégories

PromptCraft détecte automatiquement la catégorie lors de la génération. Les catégories disponibles :

Catégories
💻  Développement & Code
✍️  Rédaction & Contenu
📊  Analyse & Recherche
🎨  Créativité & Design
📧  Communication & Email
📈  Marketing & Growth
🎓  Éducation & Formation
🤖  IA & Automatisation

Favoris

Marquez un prompt comme favori en cliquant sur l'étoile ⭐. Les favoris apparaissent en tête de votre bibliothèque et sont accessibles via le filtre "Favoris" dans la barre latérale.

Partage #

Partager un prompt

Chaque prompt peut être partagé via un lien public unique :

  1. Ouvrez le prompt dans votre bibliothèque
  2. Cliquez sur "Partager" → "Générer un lien"
  3. Copiez le lien — le destinataire n'a pas besoin de compte
🔗
Les liens partagés sont en lecture seule. Vous pouvez les désactiver à tout moment depuis les paramètres du prompt.

Collaboration Pro

Avec le plan Team, jusqu'à 5 membres peuvent partager une bibliothèque commune :

  • Bibliothèque partagée en temps réel
  • Permissions par membre (lecture / édition / admin)
  • Commentaires et annotations sur chaque prompt
  • Historique des modifications avec attribution

Optimisation #

Comment fonctionne l'amélioration des prompts

PromptCraft utilise un pipeline LangGraph en 3 agents spécialisés :

A1
Agent d'Analyse
Détecte l'intention, le domaine, le ton et le niveau de complexité. Classificateur fine-tuné sur 50k+ exemples.
A2
Agent de Structuration
Génère les 5 sections (Rôle, Contexte, Tâche, Format, Contraintes) selon les meilleures pratiques de prompt engineering.
A3
Agent de Raffinement
Assemble les sections en texte fluide et cohérent. Calcule un score de qualité de 0 à 100.

Bonnes pratiques

🎯
Soyez spécifique
Mentionnez l'outil cible, le ton souhaité, le format de sortie et votre audience.
📝
Donnez du contexte
Précisez votre secteur, niveau d'expertise et l'usage final du prompt.
🔄
Itérez
Régénérez en ajoutant des précisions — chaque itération affine le résultat.
⭐
Sauvegardez les bons
Mettez en favori les prompts efficaces pour les réutiliser rapidement.

💻
